## Pre requierments

- docker
- docker-compose
- node.js 20
- pnpm
- python 3.11

## Install and Run via docker compose

- clone repository
- rename .env.exemple -> .env
- run docker-compose up from the project folder
- check in browser: localhost:2080

```
sudo docker-compose up
```
